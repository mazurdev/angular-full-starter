## :heavy_check_mark: Getting started
```bash
git clone https://github.com/mazurdev/angular-full-starter.git
cd to folder
npm run start | yarn start
```

## Build
```bash
npm run build | yarn build
cd dist
http-server
open http://localhost:8080
```
